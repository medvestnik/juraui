Jura UI

Lightweight UI kit for admin panels, dashboards, SaaS interfaces and product pages. Clean, card-based and framework-agnostic.

View examples GitHub repository Getting started PHP integration

Designed for fast interfaces

Jura UI helps ship admin tools quickly while staying readable and easy to customize.

  • Lightweight CSS architecture
  • No heavy dependencies
  • Works in static, backend and frontend projects

Core benefits

Lightweight

Small, practical CSS and HTML patterns.

Framework agnostic

Works with plain HTML, backend templates and SPA wrappers.

Admin-first

Built for CRUD interfaces, dashboards and internal tools.

Responsive

Cards, layouts and tables adapt by default.

Easy to customize

Theme tokens are exposed as CSS variables.

Open source

MIT licensed and hosted on GitHub.
